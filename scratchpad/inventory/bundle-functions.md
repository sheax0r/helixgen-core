## Key finding: global settings ARE network-addressable

The app reads/writes device global settings via `/PropertyValueGet` / `/PropertyValueSet`
over the same 2002 RPC channel, keyed by a dotted **`global.*`** namespace (251 keys).
This means the entire Global Settings surface is controllable over the network — the gap is
exposing these keys as helixgen CLI verbs, plus discovering the exact PropertyValueSet arg shape (capture).

## Command families (P35EditCommandDefs.json) — footswitch/command-center assignable commands

Families: Undefined, PresetSnapshot(+Instant/Drum), Song, Looper, Utility, ExtAmp, MIDI-Instant, MIDI-FS, MIDI-Exp, MIDI-Drum, MIDI-XY, HotKey. Each has integer `Action`/`Command` params; see e.g. Looper (Command 0-6), PresetSnapshot (Setlist/Preset/Snapshot selectors), MIDI-FS (Ch/MSB/LSB/PC/CC), Utility (Level Trim ±20dB).

### dialog_preferences.xml (app + device prefs)
Auto HW Assign, FX Pedal 1/2 Jack, Hardware Compatibility Mode, HCM File Option, per-model FS function (P33 FS3-5, P36 FS7/8/Tap), ExpFS Tip/Ring, LT EXP2/Ext Amp. (Classic HX Edit 'Global Settings > Ins/Outs' pages are NOT a separate app screen here — those live in the global.* property namespace, edited on-device or via PropertyValueSet.)

### Notable capability screens
- view_command_center — Command Center (per-footswitch command assignment; uses the command families above)
- window_assigns / popup_ctrlassign / popup_midiassign — controller + MIDI assignment
- view_globaleq — Global EQ (dsp.globaleq.*)
- widget_tempo_panel — tempo/tap (global.tempo.*, /SetTempo, /SetTimeSignature)
- view_remoteaccess / dialog_remoteaccess_pin — Line 6 Remote Access (cloud, likely out-of-scope)
- window_updatecenter / dialog_outdated_firmware — firmware update (out-of-scope)
- browser_view_ir / popup_ir_select — IR management
- browser_view_song / view_showcase_* / dialog_export_song — Song/Showcase (multitrack player) feature
